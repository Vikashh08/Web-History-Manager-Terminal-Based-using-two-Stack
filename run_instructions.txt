# How to Run the Project

This project simulates a browser's back and forward navigation using the Two Stacks data structure.

### For Mac and Linux:

1. Open your terminal.
2. Navigate to the project folder.
3. Run the following command to compile:
   ```bash
   g++ -o browser_manager main.cpp
   ```
4. Run the program:
   ```bash
   ./browser_manager
   ```

### For Windows:

**Using g++ (MinGW):**
1. Open Command Prompt (cmd) or PowerShell.
2. Navigate to the project folder.
3. Run the following command to compile:
   ```bash
   g++ -o browser_manager.exe main.cpp
   ```
4. Run the program:
   ```bash
   browser_manager.exe
   ```

**Using Microsoft Visual C++ (cl.exe):**
1. Open Developer Command Prompt for VS.
2. Run:
   ```cmd
   cl main.cpp
   ```
3. Run the program:
   ```cmd
   main.exe
   ```
